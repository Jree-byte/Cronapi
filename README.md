This project demonstrates how to fetch data from external APIs, store it in a MySQL database, and visualize it in real-time using Streamlit. The setup uses a cron job to automatically update data every 15 minutes.

Features

Fetch weather data for Helsinki using OpenWeatherMap API

Fetch aviation data (flights) from a chosen aviation API

Store both datasets in MySQL tables (weather_data and aviation_data)

Automatic updates using cron jobs every 15 minutes

Visualize latest data in Streamlit dashboard

Line charts for trends over time
